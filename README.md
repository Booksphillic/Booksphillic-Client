
## 북스필릭

동네책방과 소비자를 연결하여 서점 소개, 책 추천, 이벤트 소개까지 한번에 제공하는 동네책방 올인원 플랫폼


## 기술 스택

- 프론트엔드: Javascript, React, styled-component (recoil 적용하여 리팩토링 예정)
- 백엔드: Java, Spring Boot, Spring Data JPA, Spring Security
- DB: MySQL
- 배포: AWS ec2, elastic beanstalk
